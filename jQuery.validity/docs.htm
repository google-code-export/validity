Static Functions:

jQuery.validity.Setup(options) Use this function to setup an error output mode on load. Available settings are outputMode, RaiseCustomOutputModeError, RaiseCustomOutputModeAggregateError and CustomOutputModeClear. Their meanings are summarized below:
outputMode outputMode is a string representing which output mode you want. Accepted values are:
Modal Use this when you want an error message to float next to the offending input. If you use this a div with id validity-modal-output must be defined on the page to contain the output.
Summary Use this when you want a list of error messages in one place with the offending inputs highlighted. If you use this a div with id validity-summary-output should be defined.
Custom This will allow you to roll your own output mechanism based on how you define the other three options.
None Don't use error message outputs. Validity will only perform logic.
RaiseCustomOutputModeError Assign a custom function for raising normal error messages. The function should accept a jQuery object of the offending element and a string of the error message. This will only be used if outputMode is set to Custom.
RaiseCustomOutputModeAggregateError Assign a custom function for raising aggregate error messages. The function should accept a jQuery object of the offending elements and a string of the error message for all of them. This will only be used if outputMode is set to Custom.
CustomOutputModeClear Assign a custom function for removing all error messages. This will only be used if outputMode is set to Custom.
Simple examples:

jQuery.validity.Setup({outputMode:"Modal"});
jQuery.validity.Setup({outputMode:"Summary"});


Custom Output Mode example:

jQuery.validity.Setup({
  outputMode:"Custom",
  RaiseCustomOutputModeError:function($obj, msg){
    alert("The Problem With The Input Named: " + obj.attr('id') + " is \'" msg + "\'"); }
  },
  RaiseCustomOutputModeAggregateError:function($obj, msg){
    alert("The Problem With " + obj.length + " input(s) is \'" msg + "\'"); }
  },
  CustomOutputModeClear:function(){ }
);
jQuery.validity.Clear() Clear all existing validation errors from the page. This static function should be called before validation is begun so that there are no abandoned error messages. This function will use whatever clear mechanism is specified by the setup function (e.g. 'modal', 'summary', 'custom').
Example:

jQuery.validity.Clear();

jQuery.validity.GeneralError(msg) Post a general error message.
